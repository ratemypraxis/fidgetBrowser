# Fidget Browser

A Chrome extension that uses hand tracking to let you anxiously spin the internet! 
by Re Chang and Sim Ciudad.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right corner)
3. Click "Load unpacked"
4. Select the `fidget-browser` directory
5. Grant camera permissions when prompted

## Usage

1. Navigate to any webpage
2. Allow camera access when prompted by the browser
3. Show your hand to the camera (the feed shows in the upper right corner)
4. Move your thumb quickly to trigger the webpage spin animation
